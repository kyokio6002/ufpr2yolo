# ufpr2yolo

you can use this to convert
[raysonlaroca/ufpr-alpr-dataset](https://github.com/raysonlaroca/ufpr-alpr-dataset)
format to
[AlexeyAB/darknet](https://github.com/AlexeyAB/darknet)
format

```tree
C:.
├─ufpr2yolo.py
├─testing
│  ├─track0091
│  ├─track0092
│  ├─...
│  └─track0150
├─training
│  ├─track0001
│  ├─track0002
│  ├─...
│  └─track0060
└─validation
    ├─track0061
    ├─track0062
    ├─...
    └─track0090
```
